Reproducir el análisis

Cómo correr la simulación localmente

El código está pensado para ejecutarse en cualquier entorno Python 3 con las librerías científicas estándar.

Pasos para reproducir

  1. Descargá el ZIP completo y descomprimilo en una carpeta de trabajo.
  2. Instalá las dependencias: pip install numpy pandas scipy matplotlib
  3. Ubicate dentro de la carpeta Código/ y abrí simulacion_bass.py con tu editor.
  4. Ajustá la variable BASE_DIR al inicio del script con la ruta donde descomprimiste el ZIP.
  5. Verificá que el archivo datos_para_simulacion.csv de la carpeta Datasets/ esté accesible desde el script.
  6. Ejecutá con python simulacion_bass.py. El script crea las carpetas necesarias, ajusta el modelo, corre los cuatro escenarios y guarda las seis figuras y el resumen JSON.
  7. Tiempo total de ejecución estimado: menos de 30 segundos en una computadora moderna.
